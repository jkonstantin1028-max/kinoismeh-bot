# Telegram Bot @Kinoismeh

Автопостинг новостей, мемов и лайфстайла в канал @Kinoismeh.

## 🚀 Запуск на Railway/Render
1. Создай репозиторий на GitHub и загрузи файлы.
2. Подключи репозиторий к Railway или Render.
3. В настройках проекта добавь переменные окружения:
   - BOT_TOKEN = твой токен от BotFather
   - CHAT_ID = @Kinoismeh
4. Деплой — и бот работает 24/7.
